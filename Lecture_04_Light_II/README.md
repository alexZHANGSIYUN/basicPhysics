### 问题4　什么是实像，什么是虚像？ 

### 读一读：

当物体与透镜的距离大于焦距时，物体成倒立的像，这个像是蜡烛射向凸透镜的光经过凸透镜会聚而成的，是实际光线的会聚点，能用光屏承接，是实像．平面镜所成的像是虚像，因为在成像的位置，并没有光在那里会聚，只是我们感到光好像从那里射出的．与引类似，当物体与透镜的距离小于焦距时，物体成正立的虚像（不能用光屏承接，只能用眼睛看到）．
